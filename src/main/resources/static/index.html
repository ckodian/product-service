Welcome, The Wishlist Company!

This application is a microservice.

  • It does not have a front-end.
  • It is serving REST APIs, under the '/api' URLs.
  • To manage this microservice, you will probably want to use a Registry
  • Swagger documentation endpoint for those APIs is at /v2/api-docs, but if you want access to the full Swagger UI, you should use a TWC gateway, which will serve as API developer portals.
